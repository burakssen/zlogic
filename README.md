# zlogic

A logic gate simulator similar to Logisim, written in Zig and Raylib.

## Controls

- **1**: Switch to "Place Gate" mode.
- **2**: Switch to "Draw Wire" mode.

### Place Gate Mode
- **Left Click**: Place an AND gate on the grid.

### Draw Wire Mode
- **Left Click**: Start drawing a wire.
- **Left Click (again)**: Finish drawing the wire.

## Build & Run

```bash
zig build run
```
